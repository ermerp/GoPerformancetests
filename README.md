# Readme

## Build

Navigate to the projects root folder (/GoPerformancetests)

### Windows (Git Bash)
```
./build.sh
```
Executables:
```
.\bank\bankGo.exe
.\mergesort\mergesortGo.exe
```

### Unix
```
chmod +x build.sh
./build.sh
```
Executables:
```
./bank/bankGo
./mergesort/mergesortGo
```
